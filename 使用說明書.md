# LTC 重疊檢查工具 — 使用說明書

---

## 一、工具說明

本工具提供兩種執行方式：

| 版本 | 檔案 | 說明 |
|---|---|---|
| **Console 版** | `LTC_Checker.exe` | 命令提示字元視窗執行，自動跑完並顯示結果 |
| **UI 版** | `LTC_Checker_UI.exe` | 圖形介面，可選擇資料夾、查看進度與結果 |

---

## 二、資料夾結構

兩個版本的 exe 執行時，會在**與 exe 相同的資料夾**內尋找以下目錄：

```
LTC_Checker.exe（或 LTC_Checker_UI.exe）
csv/                  ← 服務紀錄（每月下載的 xlsx，可多個公司）
support/
    CS/               ← 各公司子資料夾（名稱自訂）
    CZ/
    YH/
    ...
class/
    CS/
    CZ/
    YH/
    ...
save/                 ← 檢查結果輸出（自動建立）
```

---

## 三、各資料夾放哪些檔案

### `csv/`
- 放各公司的**服務紀錄** xlsx（每月下載）
- **檔名不限**，工具依內容自動辨識
- 可以同時放多個公司的檔案

### `support/<公司>/ `（支援檢查）
每個公司子資料夾放 **2 個 xlsx**：
1. **支援排班表**（欄數較少，約 5~6 欄）
2. **單頁服務紀錄**（欄數較多，47 欄以上）

> **檔名不限**，不需要含 "support" 等關鍵字，工具依欄位數自動判斷。

### `class/<公司>/`（上課檢查）
每個公司子資料夾放 **2 個 xlsx + 1 個 txt**：
1. **上課排班表**（欄數較少）
2. **單頁服務紀錄**（欄數較多）
3. **參加人員名單 `.txt`**（每行一個姓名，UTF-8 編碼）

> 同樣檔名不限。

---

## 四、Console 版操作步驟

1. 依上述結構準備好資料夾與檔案
2. 雙擊執行 `LTC_Checker.exe`（或在命令提示字元輸入路徑執行）
3. 自動讀取資料並顯示進度，完成後關閉視窗即可
4. 結果存在 `save/` 資料夾內

> **注意：** 若 `support/` 或 `class/` 資料夾不存在或為空，該項目將自動跳過，不影響其他檢查。

---

## 五、UI 版操作步驟

1. 雙擊執行 `LTC_Checker_UI.exe`
2. 左側面板選擇資料夾：
   - **服務紀錄資料夾 \*** → 選 `csv/` 資料夾（必填）
   - **支援資料資料夾**（選填）→ 選 `support/` 資料夾
   - **上課資料資料夾**（選填）→ 選 `class/` 資料夾
   - **結果儲存位置 \*** → 選 `save/` 資料夾（必填）
3. 按「**▶ 開始執行**」
4. 左側下方顯示執行紀錄，進度條顯示進度
5. 右側面板顯示各項檢查結果：
   - ✓ 無問題（綠色）
   - ⚠ 有衝突（紅色），點「**開報告**」可直接開啟對應報告檔
6. 完成後可按「**開啟存檔資料夾**」查看所有結果

> 路徑設定會自動儲存，下次啟動時自動帶入。

---

## 六、結果檔案說明

```
save/
    Patient/
        <個案姓名>_patient_overlap.txt   ← 個案時間衝突紀錄
    Worker/
        <居服員>_worker_overlap_<原因>_<公司>.txt   ← 居服員衝突紀錄
    support/
        <公司>/output_support.txt        ← 超出支援範圍紀錄
    class/
        <公司>/output_class.txt          ← 上課期間有服務紀錄
```

每筆衝突紀錄包含：服務時間、個案、居服員、公司、地址、**來源檔案名稱與 Excel 列號**（方便回原檔確認）。

## 七、常見問題

**Q: 放進去的檔案需要改名嗎？**
A: 不需要。工具依檔案內容（欄位數）自動判斷，檔名不影響結果。

**Q: 每個月換資料只要替換檔案就好嗎？**
A: 是的。把舊檔刪除（或清空各子資料夾），放入新月份的下載檔案，直接執行即可。

**Q: support 或 class 資料夾沒有資料會怎樣？**
A: 工具會自動跳過，只執行有資料的項目，不會報錯。

**Q: 每個公司子資料夾的名稱（CS、CZ 等）可以自訂嗎？**
A: 可以，名稱不影響判斷，只是用來區分公司。

**Q: UI 版的路徑設定存在哪裡？**
A: 存在 exe 同目錄的 `ltc_config.json`，可手動刪除重置。
